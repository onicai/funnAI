These scripts were written specifically for this initial test configuration:

The testing, development or ic networks:
- One Challenger          with 1 LLM
- One Judge               with 1 LLM
- One ShareService        with 1 LLM
- One or more mAIner #Own with 1 LLM each
- One or more mAIner #ShareAgent

The local network:
- One Challenger          with 2 LLMs
- One Judge               with 2 LLMs
- One ShareService        with 2 LLMs
- One or more mAIner #Own with 1 LLM each
- One or more mAIner #ShareAgent

During the tests, all timers must be off, except for the ShareService.
